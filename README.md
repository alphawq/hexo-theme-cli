# hexo-theme-cli

Auto generate project template of hexo theme

## Prerequisites

- NodeJs


## Getting Started

Install:

```
 $ npm install -g hexo-theme-cli
```

You can use it by running:

```
    hexome -h
```

You can use the `-i` parameter to specify the template engine you want to use 

```
    hexome -i ejs

    # or

    hexome // `swig` is default
```
